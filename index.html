Social Bot Request Throttle

A WordPress plugin to limit request frequency from social media web crawlers

Key Features

Throttle Control

Configure throttle times for Facebook, Twitter, Pinterest, and custom bots

Customizable

Set custom user agent detection and image request throttling for each crawler

Monitoring

Comprehensive logging capabilities and built-in testing tools

Installation

1

Download the plugin code

2

Upload to /wp-content/plugins/

3

Activate through WordPress plugins menu

Configuration

Default Settings

  • Facebook: 60 seconds
  • Twitter: 60 seconds
  • Pinterest: 60 seconds

Customization Options

  • Throttle time in seconds
  • User agent strings
  • Image request throttling
